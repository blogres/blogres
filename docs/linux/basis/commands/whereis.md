⑥、语法：

whereis [选项] 文件

*选项：*

- -b         只搜索二进制文件
- -B <目录>  定义二进制文件查找路径
- -m         只搜索 man 手册
- -M <目录>  定义 man 手册查找路径
- -s         只搜索源代码
- -S <目录>  定义源代码查找路径
- -f         终止 <目录> 参数列表
- -u         搜索不常见记录
- -l         输出有效查找路径
