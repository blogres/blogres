### cd 切换目录

- [返回命令大全列表](../command.md#磁盘管理)

①、命令名称：cd

②、英文原意：change directory

③、命令所在路径：**shell 内置命令**

④、执行权限：所有用户

⑤、功能描述：切换目录

⑥、语法： cd【目录名】

**例子：**

切换到指定目录：`cd /home`

回到上一级目录：`cd ..`

还是在当前目录：`cd .`

返回上两级目录：`cd ../..`

返回进入此目录之前所在的目录：`cd -`
