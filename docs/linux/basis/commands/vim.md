vi filename，创建文件的同时写入内容

vim 是 vi 编辑器的增强

vim进入编辑文本，i开始输入，esc命令模式，： 在命令模式下输入冒号进入编辑模式，输入命令+enter回到命令模式，：wq保存退出：q不保存退出 ：q ! 强制退出:x 保存并退出

**快捷键：**

`dd` 删除当前行，`yy` 复制当前行，`p`复制当前行，`：set nu` 设置行号，`：set nonu`取消行号，`：n`到第n行，`$`移动到行尾，`0`移动到行首
