#### 引用与指向——对象变量与堆空间开辟的独立对象空间

new GirlFriend();

代表整出一个模型生产出来的对象。

意味着在堆空间中开辟了独立的运行（内存）空间，没有办法在方法中找到这个独立空间的起始地址。

为了访问到他，我们需要给一个名字变量（引用-把目标引过来用他，指针）

GirlFriend gf= new GirlFriend();

Gf就是GirlFriend的一个对象，引用，指向堆中空间

=  等号就是指向，代表这个堆中对象的起始地址
